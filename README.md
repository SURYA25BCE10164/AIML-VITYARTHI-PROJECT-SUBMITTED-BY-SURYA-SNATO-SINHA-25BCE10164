# 🍔 Food Image Classifier with Calorie Estimation

A CNN-based Machine Learning project that classifies food images and estimates their calorie content.



## Features
-  Upload a food image
-  Classifies food using a Convolutional Neural Network (CNN)
-  Displays estimated calories per serving
-  Shows prediction confidence
-  Simple and interactive UI using Streamlit



##  How It Works
1. The user uploads an image.
2. The image is preprocessed (resized, normalized).
3. A trained CNN model predicts the food category.
4. The predicted label is mapped to a calorie value using a predefined dataset.



##  Tech used for this project 
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit

## Limitations

-Calorie values are approximate
-Accuracy depends on dataset quality
-Portion size is not automatically detected
